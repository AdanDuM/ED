Detailed Description

Classe que modela uma lista encadeada.

Uma lista encadeada ou lista ligada é uma estrutura de dados linear e dinâmica. Ela é composta por células que apontam para o próximo elemento da lista. Para "ter" uma lista ligada/encadeada, basta guardar seu primeiro elemento, e seu último elemento aponta para uma célula nula.
Para mais informações acesse:
http://pt.wikipedia.org/wiki/Lista_ligada

int Projeto1::ListaCircular::adiciona ( Caixa *  caixa)

Método que adiciona no início um objeto da classe ElementoDeLista.

Para isso verifica se ha espaço na mesma. Caso a questão anterior seja verdadeira o objeto é adicionado na primeira posição da Lista.

Parameters:
caixa: ponteiro para o caixa onde o cliente vai ser inserido.
Returns:

1 = inteiro que informa a posição onde o dado foi inserido.
