Detailed Description

Classe para representação de estruturas de dados Pilha.

As pilhas são estruturas baseadas no princípio LIFO (last in, first out), na qual os dados que foram inseridos primeiros na pilha serão os últimos a serem removidos. Existem duas funções que se aplicam a todas as pilhas: EMPILHA (PUSH), que insere um dado no topo da pilha, e DESEMPILHAR (POP), que remove o item no topo da pilha.

Disco * PilhaEncadeada::Pilha::desempilha ( )

Método que retira o ultimo elemento pilha.

Para isso ele verifica se a pilha esta vazia. Caso não esteja, retira o elemento que esta no topo da pilha(ultima poscição), atualiza o topo e o diametro do Topo(caso acabe vazia o diametro será considerado zero), e o retorna para ser usado.

Returns:
PILHA_VAZIA = caso a pilha em execução esteja vazia, será retornado o codigo de erro PILHA_VAZIA.
Ponteiro para um objeto do tipo Disco que está na ultima posição da Pilha.
