Detailed Description

Classe que modela uma Fila Encadeada.

As filas são estruturas baseadas no princípio FIFO (first in, first out), em que os elementos que foram inseridos no início são os primeiros a serem removidos. Uma fila possui duas funções básicas: Enfileirar(ENQUEUE), que adiciona um elemento ao final da fila, e Desenfileirar(DEQUEUE), que remove o elemento no início da fila. A operação Desenfileirar(DEQUEUE) só pode ser aplicado se a fila não estiver vazia, causando um erro de underflow ou fila vazia se esta operação for realizada nesta situação.

Para mais informações acesse: http://pt.wikipedia.org/wiki/FIFO
